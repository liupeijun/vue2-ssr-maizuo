# vue2-ssr-maizuo

### Vue的服务端渲染, 解决单页面应用的首屏加载速度和 seo等问题, 此项目是基于vue2, vuex2, vue-router2 的server side rendering的最佳实践

## Node 版本要求

```bash
node 6.*
node 7.*
```

## Build 设置

``` bash
npm install
npm run build
npm start
```

## Development 设置

```bash
npm install
npm run dev
```
